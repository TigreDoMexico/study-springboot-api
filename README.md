# Superheroes API

## Content
Rest API created with SpringBoot and Database MySQL

## How to Run Locally
Just run the following commands:

```bash
docker compose up -d

mvn clean install
mvn spring-boot:run
```

All routes are in [request.http](./requests.http) file.
